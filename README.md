# 🎓 Student Management System

A console-based Student Management System 
developed using Python. It provides functionalities 
such as adding students, viewing records, searching, 
deleting, and finding the topper through a 
menu-driven interface.

---

## 📌 Project Overview

The Student Management System helps manage student 
records efficiently. The program uses Object-Oriented 
Programming concepts to store and manipulate student 
information such as name, roll number, marks, 
average, and pass/fail status. Data is persistently 
stored in a text file.

---

## 🎯 Objectives

- Implement OOP concepts in Python
- Apply file handling for persistent data storage
- Handle exceptions gracefully
- Build a real-world console application
- Improve logical thinking and problem-solving skills

---

## 🧰 Features

- Add new student with validation
- View all students with pass/fail summary
- Search student by roll number
- Delete student record
- Display topper (highest average)
- Menu-driven user interface
- Persistent data storage using text file

---

## 🧠 Programming Concepts Used

- Object-Oriented Programming (OOP)
- Encapsulation — private attributes
- @property decorator — getters
- __str__ method — clean object display
- File Handling — read, write, append
- Exception Handling — ValueError, FileNotFoundError
- Functions and Modular Programming
- Loops and Conditional Statements

---

## ⚙️ Technologies Used

- Language: Python 3
- IDE: VS Code
- Storage: Text file (Database.txt)

---

## 📂 Repository Structure

- `main.py` — main source code
- `Database.txt` — auto-created when first student is added

---

## 🚀 How to Run

1. Make sure Python 3 is installed
2. Clone or download the repository
3. Run the program:
   python main.py
4. Use the menu to manage student records

---

## 📊 Menu Options

1. Add Student
2. View All Students
3. Search Student by Roll Number
4. Delete Student
5. Show Topper
6. Exit
